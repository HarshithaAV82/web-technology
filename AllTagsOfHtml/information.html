Header tag defines the top section of a webpage. It is usually used to contain the website title, logo, navigation menu, or introductory content.

Main tag contains the main content of the entire webpage. It is usually used only once in a page and placed inside the body. It holds the important content like projects, about, blog, or main information of the website.

Section tag divides the webpage content into different sections. You can use many section tags in a page. It is usually placed inside main or other parts of the webpage and is used for smaller parts like About, Skills, or Projects.

Article tag represents independent content of a webpage. It usually includes things like blog posts, news articles, product details, forum posts, or any content that can exist independently.

Aside tag represents side content related to the main content. It usually includes tips, advertisements, related links, side notes, or additional information placed beside the main content.


Footer tag defines the bottom section of a webpage. It usually contains information like copyright, contact details, links, or author information.
